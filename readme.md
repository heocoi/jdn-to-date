# jdn-to-date

> Converting Julian Day Number (JDN) to Date.

## Install

```sh
npm install jdn-to-date
```

## Usage

```js
import jdnToDate from "jdn-to-date";

jdnToDate(2460370);
//=> new Date('2024-02-29')
// UTC
```

## API

### jdnToDate(jdn)

#### jdn

Type: `number`

Julian Day Number of the day.
